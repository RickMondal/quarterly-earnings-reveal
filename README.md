# Quarterly Earnings RevealJS Presentation

Interactive stakeholder briefing built with [Reveal.js](https://revealjs.com/) and published via GitHub Pages.

## Local preview

```bash
npx serve .
```

Then open `http://localhost:3000` in your browser. Slides auto-refresh when the page reloads.

## Key features

- Markdown-driven KPI slide for quick edits.
- Fragments for staged storytelling.
- Syntax-highlighted code sample illustrating automation.
- Financial math rendered with KaTeX.
- Speaker notes to guide the live narrative.

## Deployment

The presentation is hosted on GitHub Pages from the `main` branch root. Append `?v=1`, `?v=2`, etc. to bypass browser caches after updates.
